A new Flutter project.

## Gemini API Key

The app reads the Gemini key from `GEMINI_API_KEY`.

Use it at launch like this:

```bash
flutter run --dart-define=GEMINI_API_KEY=your_key_here
```

If you are running from Android Studio or VS Code, add the same `--dart-define`
to the run configuration arguments instead of placing the key directly in code.
